== Description ==

Este sencillo plugin es un widget que muestra una lista de los mejores juegos de BigFish en la barra lateral Widgetized en idioma Espanol solamente (.es).
Es muy util si desea crear un sitio para diferentes idiomas y desea utilizar todas las tapas de los juegos para diferentes idiomas.

ENGLISH:
This simple plugin is a widget that displays a list of top casual games from BigFish on your widgetized sidebar on Espanol language only (.es).
It's very useable if you want to create one site for different languages and want to use all tops of games for different languages.

== Installation ==

1. Subir carpeta flash contiene 'widget-top-bfg-games-ES.php' al '/wp-content/plugins/' directorio.
2. Activar el plugin a traves del menu 'Plugins' en WordPress.
3. Utilice su 'Presentation'/'Sidebar Widgets' apartado lateral para colocar el flash y para configurarlo.

ENGLISH:
1. Upload widget folder contains 'widget-top-bfg-games-ES.php' to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use your 'Presentation'/'Sidebar Widgets' section to place the widget and to configure it.

== Configuration ==

* Widget titulo: el titulo del widget
* Numero de Juegos de Exhibicion: Numero de los mejores juegos que le gustaria que se muestra (1-10)
* Juegos Tipo de Pantalla: Tipos de juegos que usted quiere que se muestre (Juegos para PC / Juegos en linea)
* Idioma Juegos de Pantalla: Idioma juegos que usted quiere que se muestre (Ingles / Aleman / Frances / Espanol)
* Texto e imagenes Alineacion: Alineacion de la lista de juegos que usted quiere que se muestre (por defecto el tema / izquierda / centro / derecha)
* Mostrar las vinetas en las listas: ?Es necesario para mostrar vinetas en las listas (usar el estilo de tema)
* Mostrar la posicion de Juegos: ?Es necesario para mostrar la posicion en la lista de los mejores juegos
* Mostrar imagenes Juegos: ?Es necesario para mostrar imagenes de juegos, ademas de titulo del juego
* Imagenes: Tipo de imagenes que le gustaria que se mostrara (pequeno 60x40 / default 80x80 / 175x150 aparece / subfeatured 175x150)
* Mostrar texto Juegos: ?Es necesario mostrar el texto de los juegos, ademas de titulo del juego y la imagen
* ?Que Arriba: ?Que Sobre que le gustaria que se muestra - texto o imagen
* Su nombre de usuario BigFishGames: Puede dejar este campo vacio, flash puede trabajar sin conexion, o el uso de conexion adecuados.
* El canal de cliente BigFishGames: nombre de dominio se utiliza por defecto. Deje este campo vacio, a no utilizar los canales. Alfa numericos.

ENGLISH:
* Widget title: the title of the widget
* Number of Games to Display: Number of top games you would like to be displayed (1-10)
* Games Type to Display: Games Types you would like to be displayed (PC games / Online Games)
* Text and Images Align: Align of games list you would like to be displayed (theme default / left / center / right)
* Show List Bullets: Is need to show list bullets (use theme style)
* Show Games Position: Is need to show games position in top list 
* Show Games Images: Is need to show games images in addition to game title
* Images Type: Images Type you would like to be displayed (small 60x40 / default 80x80 / featured 175x150 / subfeatured 175x150)
* Show Games Text: Is need to show games text in addition to game title and image
* What Above: What Above you would like to be displayed - Text or Image
* Your BigFishGames login: You can leave this field empty, widget can work without login, or use correct login.
* Your BigFishGames client channel: Domain name will be used by default. Leave this field empty, to not use channels. Alpha numeric only.
